# inputs/

The pipeline files dated research digests here as `YYYY-MM-DD-<slug>.md`:

- `*-signal-digest.md` — squeeze + momentum watchlist (`flab-signals`)
- `*-market-divergence.md` — cross-venue Kalshi/Polymarket gaps (`flab-divergence`)
- `*-arxiv-triage.md`, `*-faculty-fit.md`, … — research sweeps (see `research-sources.md`)

Generated notes are gitignored (see `.gitignore`); this folder and its README are kept.
Drop this directory into an Obsidian vault and the digests link in with `[[brackets]]`.
